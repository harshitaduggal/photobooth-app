# photobooth-app
A web-based AI photobooth application that allows users to capture or upload photos, apply cute aesthetic filters and stickers, and enhance portraits using computer vision, all through an interactive Streamlit interface.
 Features

 Camera capture & image upload

Aesthetic filters
(Vintage, Soft Glow, Pastel, Black & White)

 Sticker overlays
(PNG-based, resizable & positionable)

 Face detection–based enhancements
